# Display package versions

Print version information for
[tiledb](https://CRAN.R-project.org/package=tiledb) (R package),
libtiledbsoma, and TileDB embedded, suitable for assisting with bug
reports.

## Usage

``` r
show_package_versions()
```

## Examples

``` r
show_package_versions()
#> tiledbsoma:    2.3.0
#> tiledb-r:      0.34.0
#> tiledb core:   2.30.0
#> libtiledbsoma: 2.30.0
#> R:             R version 4.5.2 (2025-10-31)
#> OS:            Ubuntu 24.04.3 LTS
```
